1. position: absolute vs relative

- relative: phần tử giữ vị trí ban đầu, nhưng có thể dịch chuyển.
- absolute: phần tử thoát khỏi luồng và định vị theo phần tử cha gần nhất có position. 

.parent { position: relative; }
.child { position: absolute; top: 0; right: 0; }

Responsive: absolute dễ bị lệch trên màn nhỏ → cần dùng flex/grid.

3. var vs let vs const

var a = 1;
let b = 2;
const c = 3;

- var: function scope, hoisting
- let: block scope
- const: không đổi

Lỗi thường gặp:

console.log(a);
let a = 5; // lỗi

Debug: dùng Chrome DevTools (F12 → Console)
